The response you will receive for getting the profile for all users or for one specific user has the same structure except that the array with profile objects in the response for one specific user will only contain one object for that specific user.

Response structure
This is an overview of all fields that can be inside an user profile object. When a field is empty or not public it will not be visible in the response.

Field Type Description
count Number The number of items.
items Object[] A list of projects (Array of Objects)
type Object Type of the research item, 4:competitive, 5:industrial
code String The code of the project
acronym String The acronym of the project
title String The title of the project
abstract String The abstract of the project
role String The role of the institute in this project
status String The status of the project
url String The url of the project in the administrative software
logos Object[] The array of logos
name String Name of the logo
description String Description of the logo
image String Base64 encoded image of the logo
startDate Date The start date of the project
endDate Date The end date of the project
projectType String The official type of the project
pi Object[] The list of pi of the project
email String The email of the pi
name String The name of the pi
surname String the surname of the pi
lines Object[] The list of the research lines involved
code String The code/CDR of the research line
description String The name of the research line
