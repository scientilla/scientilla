The response you will receive for getting the profile for all users or for one specific user has the same structure except that the array with profile objects in the response for one specific user will only contain one object for that specific group.

Response structure
This is an overview of all fields that can be inside a group profile object. When a field is empty or not public it will not be visible in the response.

Field Type Description
count Number The number of items.
items Object[] A list of group profiles (Array of Objects)
description String Description of the group
achievements String Achievements of the group
collaborations String Collaborations of the group
laboratories String Laboratories of the group
url String URL to a specific website of the group
topics String[] List of topics names (Array of Strings)
coverImage String URL of the group cover image
id Number The group's research entity ID
active Boolean The group is a current active group (true) of IIT or not (false).

Example
An example of a public profile can be like: 

    {
    "count": 1,
    "items": [
        {
            "description": "<p>This is a description</p>",
            "achievements": "<p>This is a text about achievements</p>",
            "collaborations": "<p>This is a text about collaborations</p>",
            "laboratories": "<p>This is a text about laboratories</p>",
            "url": "https://www.iit.it/",
            "topics": [,
                "Topic 1",
                "Topic 2",
                "Topic 3",
            ],
            "coverImage": "api/v1/groups/slug/istituto-italiano-di-tecnologia/cover-image",
            "id": 1,
            "active": true,
        }
    ]
}
